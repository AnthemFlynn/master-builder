Left-Click: destroy block
Right-Click: place block

Material Selection (2 methods):
  Method 1: C/Tab → Category letter → Number
  Method 2: Shift+Letter → Number
  G=Ground, S=Stone, W=Wood, I=Light, M=Metals, T=Glass
  Example: I→2 or Shift+I→2 = Redstone Lamp
Quick Select: 1-9 keys (current inventory)

Movement:
WASD: move
Space: jump / fly up
Shift: sneaking / fly down
Q: toggle walking / flying mode

Other:
C or Tab: toggle category mode
ESC: exit category mode
E: menu
F: fullscreen

Render Distance: 3

Field of View: 70

Music: On
